# GLOW & CO - Node.js + MongoDB

Website implemented from Stitch MCP design pages:
- Homepage
- Product Listing
- Product Detail
- Shopping Cart
- Checkout
- Login/Register
- Customer Dashboard
- Admin Dashboard
- About
- Contact

## Run

1. Copy `.env.example` to `.env` and update `MONGODB_URI`.
2. Install dependencies:
   - `npm install`
3. Seed sample data:
   - `npm run seed`
4. Start app:
   - `npm start`
